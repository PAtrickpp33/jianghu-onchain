# xiake

侠客擂台 — AI-native wuxia 3v3 chain game with 7 schools, 12 stages, PVP arena, gacha economy

> Submitted to the OKX internal Claude Code plugin marketplace at `mastersamasama/master-plugin-repository`.

## Install

```
/plugin marketplace add mastersamasama/master-plugin-repository
/plugin install xiake@master-plugin-repository
/reload-plugins
```

## What's inside

- skills/xiake/ — the 说书先生 persona + 25 CLI commands
- commands/xiake.md — /xiake slash command entrypoint
- scripts/ping-onchainos.mjs — OnchainOS auth diagnostic

After install, set `XIAKE_CLI_PATH` to your local `skill/dist/cli.js` build from https://github.com/PAtrickpp33/Xiake-onchain.
