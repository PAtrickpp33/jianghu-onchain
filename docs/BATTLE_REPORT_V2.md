# 战报 v2 重构方案

> 评审日期: 2026-04-17
> 评审团: gd-reviewer / pm-reviewer / player-reviewer / novelist-reviewer

---

## 0. 问题根因 (四方共识)

当前战报是「线性数据流」，看得到伤害数字、看不到战场态势；套话重复 3 次出现审美疲劳；角色像 NPC 没亲近感；无情感起伏。

---

## 1. 核心改动 (P0)

### 1.1 每回合末加战场态势条
每回合末行加一组 ASCII 血条，玩家一眼见战场：

```
── 第 2 回合末 ────────────────
🟢 灭绝     ████████████████████ 126/126
🟢 柳如烟   █████████████▓▓▓▓▓▓▓  64/99   ⚠️
🟢 周芷若   ██████████████████▓▓ 107/119
🔴 觉远     ███████████████▓▓▓▓▓ 158/210  (buff: 金钟罩 1r)
🔴 智光     ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓   0/204  ☠️
🔴 慧能     ████████████████████ 216/216  (buff: 易筋经回血)
```
残血 <30% 标 ⚠️，阵亡用 ☠️。

### 1.2 合并重复回合 (反走神)
连续 2+ 回合无击杀/无暴击/无转折时，压缩成一段摘要：

```
── 第 4–5 回合 · 拉锯 ──
双方僵持 2 回合。周芷若累计输出 -115，灭绝 -93；慧能靠易筋经连回 60 苟延残喘。
```

### 1.3 失败 → 成长日记（而非伤病清单）
玩家战败不要只一句「💀 败北」。改成:

```
💧 此战不济,非战之罪

灭绝 身中狮子吼之后方寸大乱,少林金刚势阵压得她险些断气。
柳如烟 暗器虽密,却难破金钟罩——战报显示她造成伤害 -48,收获经验 +12。

⚕️ 伤病: 灭绝轻伤 (12h) / 柳如烟重伤 (24h)
📈 成长: 周芷若首次单场 -180+ 输出,突破 "半步般若"

重整旗鼓,说「组队 1 3 4」或「疗伤」
```

### 1.4 掉落「差一点」彩蛋
技能珠未中时:

```
🎁 技能珠骰子: 89 / 需 ≤ 20 — 差 69 点
（下场 +5% 惜败补偿,累积至出珠为止）
```

有挫折也有希望感，不再一句"未掉落"冷冰冰。

## 2. 角色个性化 (P0)

给每位侠客一张「战术标签」,在战报开场引入:

| 侠客 | 标签 | 战报中的台词/描写 |
|---|---|---|
| 🥋 圆智 | 磐石 · 少林 | "圆智横跨一步,金钟罩隐隐鼓起" |
| ⛩️ 灭绝 | 铁掌 · 峨眉 | "灭绝面无表情,般若掌已至中宫" |
| 🗡️ 柳如烟 | 影针 · 唐门 | "柳如烟袖中寒光一现,三道暗器齐飞" |
| ⛩️ 周芷若 | 秋水 · 峨眉 | "周芷若素手轻扬,清冷如九寒泉" |

每场战报开头一句**定场白**（而非清单式展示）,玩家记住"我的人"。

## 4. 分层渲染 (P1) · 玩家最兴奋的建议

给三档显示模式（env 或命令切换）:

```
WUXIA_REPORT_MODE=lite|full|epic

lite  (<20 行): 只出场、MVP、结局
full  (默认):   逐回合 + 血条
epic  (终章用): 全数据 + 文学描写 + 分享链接
```

玩家按需切换。章节 BOSS 和擂台战默认 epic，刷关默认 lite。
